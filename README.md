# Primeiro Projeto Django

Este é o meu primeiro projeto Django, desenvolvido com o objetivo de aprender e entender como as coisas funcionam nesse framework web poderoso. O projeto consiste na criação de um sistema básico para cadastro de clientes e produtos.

## Funcionalidades

1. Cadastro de Clientes: O sistema permite o cadastro de novos clientes, fornecendo campos como nome, endereço, e-mail e telefone.

2. Cadastro de Produtos: É possível cadastrar novos produtos, fornecendo informações como nome, descrição e preço.

## Configuração do Ambiente

Para executar este projeto em seu ambiente local, siga as etapas abaixo:

1. Certifique-se de ter o Python instalado. Você pode fazer o download do Python em [python.org](https://www.python.org/downloads/).

2. Clone este repositório em seu ambiente local ou faça o download do código-fonte.

3. No diretório raiz do projeto, instale as dependências do projeto executando o seguinte comando:
   
pip install -r requirements.txt

4. Execute as migrações do Django para criar o banco de dados:

python manage.py migrate


5. Inicie o servidor de desenvolvimento do Django:

python manage.py runserver


6. Acesse o sistema em seu navegador através do seguinte endereço: [http://localhost:8000](http://localhost:8000)
